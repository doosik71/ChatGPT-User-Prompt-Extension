# Notes

## Zip file name

ChatGPT-User-Prompt-Extension-v1.5.0.zip

## Notes for Certification

This extension adds a prompt management panel to ChatGPT pages (<https://chatgpt.com/>). It lets users save, edit, reorder, delete, and reuse custom prompts stored locally in the browser. The extension injects a small UI into the page so users can open the prompt panel, select a saved prompt, and paste it into the ChatGPT input area. It also supports simple template placeholders such as {{text}}, {{clipboard}}, and {{selection}}.

All prompt data and UI preferences are stored only in the browser’s local storage. No user data is transmitted to any external server, and the extension does not include analytics, advertising, remote code execution, or background network communication. Required permissions are limited to activeTab and storage, and web-accessible resources are limited to local extension HTML/CSS/JSON files used to render the UI.

Recent updates improved the user interface, including draggable positioning for the menu button and resizable prompt panel width, with those preferences also saved locally for convenience

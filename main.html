ChatGPT User Prompt

(ver )

Open the prompt panel inside ChatGPT, manage saved prompts, drag the menu button up or down, and resize the panel from its left edge.

Icon/Key Description
Toggle the prompt panel open or closed
, F9 Paste and execute the current prompt
Delete the selected prompt
Update the selected prompt
Add a new prompt
Drag Drag the menu button vertically to move it
Resize Hover the panel and drag its left edge to resize it
